GenerateObject(Type) Generates an object for a given type. The type needs to be public, have a public default constructor and settable public properties/fields. Currently it supports the following types: Simple types: int, string, Enum, DateTime, Uri, etc. Complex types: POCO types. Nullables: Nullable<T>. Arrays: arrays of simple types or complex types. Key value pairs: KeyValuePair<TKey, TValue> Tuples: Tuple<T1>, Tuple<T1, T2>, etc Dictionaries: IDictionary<TKey, TValue> or anything deriving from IDictionary<TKey, TValue>. Collections: IList<T>, IEnumerable<T>, ICollection<T>, IList, IEnumerable, ICollection or anything deriving from ICollection<T> or IList. Queryables: IQueryable, IQueryable<T>.
